## 写一下 koa 里的 compose

首先说下 koa 里的 compose 函数是干啥用的，洋葱模型，一种中间件处理方式

```js
function compose(md) {
  return (ctx, next) => {
    return dispatch(0);
    function dispatch(i) {
      const fn = md[i];
      if (!fn) return Promise.resolve();
      if (i === md.length) fn = next;
      try {
        return Promise.resolve(fn(ctx, dispatch.bind(null, i + 1)));
      } catch (e) {
        return Promise.reject(e);
      }
    }
  };
}
```

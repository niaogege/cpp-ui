工具函数介绍

### 工具函数

- debounce
- throttle
- loadResource(动态加载 js/css 文件)
- observe (使用 IntersectionObserver 监视 dom 元素在文档视口的可见性)
- unobserve (取消 observe 监控)
- 埋点函数

### 常量

- isBrowser (是否是浏览器)
- isMobile (是否是移动端)
- isWeixin
